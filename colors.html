Colour Palette

Colours plays a crucial role in software development, influencing both aesthetics and usability. A well-chosen colour palette enhances user experience by improving readability, guiding user interactions, and establishing visual hierarchy. In UI/UX design colours can evoke emotions, communicate brand identity, and provide accessibility for users with visual impairments. Consistent use of colours also helps users navigate interfaces intuitively, reducing cognitive load.

TailwindCSS offers a wonderful colour palette with plenty to choose from. Every colour in the palette includes 11 steps, with 50 being the lightest and 950 being the darkest.With such a versatile range of colours at your fingertips, you can create visually stunning, accessible, and intuitive designs. Colour Utilities

Using colour utilities is very straight forward, for example, bg-teal for teal coloured background or have your text in orange by using text-orange-700 to set colour properties. To target the dark mode simply use dark variable, for example, dark:text-white-500.

  • bg-*

    The background color of an element

  • text-*

    The text color of an element

  • decoration-*

    The text decoration color of an element

  • border-*

    The border color of an element

  • outline-*

    The outline color of an element

  • shadow-*

    The color of box shadows

  • inset-shadow-*

    The color of inset box shadows

  • ring-*

    The color of ring shadows

  • inset-ring-*

    The color of inset ring shadows

  • accent-*

    The accent color of form controls

  • caret-*

    The caret color in form controls

  • fill-*

    The fill color of SVG elements

  • stroke-*

    The stroke color of SVG elements

Colour Opacity

You can change the opacity of a chosen colour using syntax such as bg-emerald/50, in this case '50' sets the alpha channel of the colour to 50%:

As a handy option this syntax also supports arbitrary values and the CSS variable shorthand: 

  • bg-white-300/[65.75%]
  • bg-teal-700/(--custom-teal)
