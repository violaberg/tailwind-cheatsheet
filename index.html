Getting Started

Adding Tailwind to your project has become a 2 second job. Few steps to follow from docs (https://tailwindcss.com/docs/installation/using-vite) and you're good to go.

Example using the easiest way Tailwind CLI:

First thing first, install tailwindcss and tailwindcss/cli:

npm install tailwindcss @tailwindcss/cli

Add import to the top of your main css (input.css in this case) file:

@import "tailwindcss";

Run the CLI tool to scan source files for classes and build your CSS (make sure you check your file names and paths):

npx @tailwindcss/cli -i ./src/input.css -o ./src/output.css --watch

or for Django run (check if static folder is in the root):

npx tailwindcss -i ./static/css/input.css -o ./static/css/output.css --watch --content './templates/**/*.html'

Add compiled CSS file to head of your html file.

<link href="./output.css" rel="stylesheet">

Congrats, you have finished setting up Tailwind v4.0

For other setup ways head over to Tailwind website.

Adding custom styles

Since Tailwind v4.0 does not use a config file by default, you can override styles directly inside your main css file using @layer

@layer base → For modifying default styles like h1, p, etc.

@layer components → For reusable custom classes like .btn-primary.

@layer utilities → For adding extra utility classes.
